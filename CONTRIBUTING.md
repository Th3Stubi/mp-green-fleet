| Element                | Abkürzung (Präfix) | Beispiel für eine gute ID   |
|------------------------|--------------------|-----------------------------|
| Button                 | btn_               | btn_start oder btn_login    |
| TextView (Text)        | tv_                | tv_welcome oder tv_headline |
| EditText (Eingabefeld) | et_                | et_email oder et_password   |
| ImageView (Bild)       | iv_                | iv_logo oder iv_profile_pic |
